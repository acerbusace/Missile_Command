Author 
------
Alex Patel and Sriram Madhusudhan

Description
-----------
Recreaction of the popular arcade game Missile Command.
The "source_code" folder contains the eclipse version of the game.

Instructions 
------------
1) Download and extract the zip file
2) Launch the "Missile_Command.jar" file

Note
----
Make sure the directory structure is the same as the one in the zip file. 
"source_code" folder is not needed to run the program.
